Sobre o Projeto

Este projeto permite a análise de dados de viaturas a partir de planilhas Excel importadas via GitHub Actions e integradas ao MongoDB Atlas. Visualize relatórios detalhados e gráficos interativos.

Desenvolvido com Flask, MongoDB Atlas, Pandas, e Tailwind CSS.
